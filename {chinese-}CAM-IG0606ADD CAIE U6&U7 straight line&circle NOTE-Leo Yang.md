# Topic 6: 直线图 (Straight-line Graphs)

## 6.1 线段的长度与中点

### 长度公式
已知两点 \(A(x_1, y_1)\)、\(B(x_2, y_2)\)，线段 \(AB\) 的长度为  
\[
AB = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}
\]

### 中点公式
中点 \(M\) 的坐标为  
\[
M\left( \frac{x_1 + x_2}{2},\; \frac{y_1 + y_2}{2} \right)
\]

**坑**：使用长度公式时，顺序无关，平方后开方取正。

---

## 6.2 平行与垂直

### 平行
若两直线斜率存在，则 \(m_1 = m_2\)。  
若一条水平 (\(m=0\))，另一条也水平。

### 垂直
若两直线斜率存在，则 \(m_1 m_2 = -1\)。  
特殊：一条水平 (\(m=0\))，一条垂直（斜率不存在）也垂直。

**几何直觉**：平行 → 方向相同；垂直 → 斜率乘积 -1 或一水平一竖直。

---

## 6.3 直线方程

### 常见形式
- **斜截式**：\(y = mx + c\)（\(m\) 斜率，\(c\) 纵截距）
- **点斜式**：\(y - y_1 = m(x - x_1)\)
- **两点式**：由两点先求斜率 \(m = \frac{y_2 - y_1}{x_2 - x_1}\)，再代入点斜式。
- **一般式**：\(Ax + By + C = 0\)（可求斜率 \(-A/B\)，截距 \(-C/B\) 等）

**步骤**：
1. 确定已知条件（一点 + 斜率；两点；斜截距等）。
2. 选用适当形式写出方程。
3. 化为所需形式（如 \(y = mx + c\) 或 \(Ax + By + C = 0\)）。

**坑**：垂直直线斜率不存在，方程应写为 \(x = a\)。

---

## 6.4 直线形面积（坐标系中的多边形面积）

### 三角形面积（已知三点）
设 \(A(x_1,y_1), B(x_2,y_2), C(x_3,y_3)\)，则  
\[
S = \frac{1}{2} \left| x_1(y_2 - y_3) + x_2(y_3 - y_1) + x_3(y_1 - y_2) \right|
\]  
或用行列式：
\[
S = \frac{1}{2} \left| \det \begin{pmatrix} x_1 & y_1 & 1 \\ x_2 & y_2 & 1 \\ x_3 & y_3 & 1 \end{pmatrix} \right|
\]

**步骤**：
1. 将三点坐标代入公式。
2. 计算绝对值，除以 2。

**示例**：\(A(1,2), B(4,5), C(6,1)\)  
\[
S = \frac12 |1(5-1) + 4(1-2) + 6(2-5)| = \frac12 |4 -4 -18| = \frac12 \times 18 = 9
\]

**坑**：三点共线时面积为 0，可用于判断共线。

### 多边形面积（可分割为三角形，或鞋带公式）

**鞋带公式**（适用于顶点按顺序排列）：
\[
S = \frac12 \left| \sum_{i=1}^{n} (x_i y_{i+1} - x_{i+1} y_i) \right|,\quad (x_{n+1},y_{n+1}) = (x_1,y_1)
\]

---

## 6.5 从非线性方程转换到线性形式

### 常用模型及转换

| 原方程 | 变换 | 新变量 | 线性形式 |
|--------|------|--------|----------|
| \(y = kx^n\) | 取 \(\lg\) | \(Y = \lg y,\; X = \lg x\) | \(Y = nX + \lg k\) |
| \(y = k b^x\) | 取 \(\lg\) | \(Y = \lg y,\; X = x\) | \(Y = (\lg b)X + \lg k\) |
| \(y = e^{px+q}\) | 取 \(\ln\) | \(Y = \ln y\) | \(Y = px + q\) |
| \(y = \frac{a}{x} + b\) | 设 \(X = \frac{1}{x}\) | \(Y = y\) | \(Y = aX + b\) |
| \(y = a\sqrt{x} + b\) | 设 \(X = \sqrt{x}\) | \(Y = y\) | \(Y = aX + b\) |

**步骤**：
1. 根据形式确定变换。
2. 计算新变量值。
3. 画图（或计算）得到直线斜率与截距。
4. 反推原方程参数。

**坑**：取对数要求数据 \(>0\)；变换后直线斜率和截距对应原方程中的参数。

---

## 6.6 从线性形式反推非线性方程

已知变量 \(X, Y\) 满足 \(Y = mX + c\)，且 \(X, Y\) 与原变量关系已知，求原方程。

### 常见反推表

| 线性形式 | 原变量关系 | 反推公式 |
|----------|------------|----------|
| \(Y = mX + c\)，\(Y=\lg y,\; X=\lg x\) | \(y = 10^c \cdot x^m\) | 幂函数 |
| \(Y = mX + c\)，\(Y=\lg y,\; X=x\) | \(y = 10^c \cdot (10^m)^x\) | 指数函数 |
| \(Y = mX + c\)，\(Y=\ln y,\; X=x\) | \(y = e^c \cdot e^{mx}\) | 指数函数（底 \(e\)） |
| \(Y = mX + c\)，\(Y=y,\; X=\frac{1}{x}\) | \(y = \frac{m}{x} + c\) | 反比例 |
| \(Y = mX + c\)，\(Y=y,\; X=\sqrt{x}\) | \(y = m\sqrt{x} + c\) | 平方根型 |

**注意**：系数 \(m, c\) 由直线读出，反推时注意底数（10 或 e）和定义域。

---

## 6.7 从数据中寻找关系

### 方法
1. 绘制原始数据的散点图，观察趋势（直线、曲线、指数、幂函数）。
2. 猜测模型（如 \(y = kx^n\)、\(y = k b^x\)、\(y = a/x + b\) 等）。
3. 对猜测模型进行变换，得到新变量 \(X, Y\)。
4. 计算新变量，再绘制散点图，检查是否近似直线。
5. 若成直线，用直线上两点（或最小二乘法概念）求斜率 \(m\) 和截距 \(c\)。
6. 反推出原方程中的常数。

**示例**：数据点 \((x,y)\) 怀疑 \(y = kx^n\)，计算 \(X=\lg x,\; Y=\lg y\)，若散点近似直线，则 \(n\) 为斜率，\(\lg k\) 为截距。

**坑**：数据误差会导致点不完全在直线上，应画“最佳拟合直线”；取对数时注意零值或负值需排除。

## 6.8 向量基础与几何应用（补充）

### 6.8.1 向量表示
- 列向量：\(\begin{pmatrix} x \\ y \end{pmatrix}\)
- 有向线段：\(\overrightarrow{AB}\)
- 单位向量组合：\(a\mathbf{i}+b\mathbf{j}\)，其中 \(\mathbf{i}=\begin{pmatrix}1\\0\end{pmatrix},\ \mathbf{j}=\begin{pmatrix}0\\1\end{pmatrix}\)

**注意**：考试中要求使用正确记号。

---

### 6.8.2 位置向量与单位向量
- **位置向量**：点 \(P(x,y)\) 的位置向量 \(\overrightarrow{OP}=\begin{pmatrix}x\\y\end{pmatrix}\)。
- **单位向量**：\(\hat{\mathbf{a}}=\dfrac{\mathbf{a}}{|\mathbf{a}|}\)，大小为 1。

**示例**：\(\mathbf{a}=\begin{pmatrix}3\\4\end{pmatrix}\)，\(|\mathbf{a}|=5\)，\(\hat{\mathbf{a}}=\begin{pmatrix}3/5\\4/5\end{pmatrix}\)。

---

### 6.8.3 向量的运算与几何应用
- **加法**：\(\begin{pmatrix}a\\b\end{pmatrix}+\begin{pmatrix}c\\d\end{pmatrix}=\begin{pmatrix}a+c\\b+d\end{pmatrix}\)
- **减法**：\(\overrightarrow{AB}=\overrightarrow{OB}-\overrightarrow{OA}\)（终点减起点）
- **数乘**：\(k\begin{pmatrix}a\\b\end{pmatrix}=\begin{pmatrix}ka\\kb\end{pmatrix}\)
- **模**：\(|\mathbf{a}|=\sqrt{a^2+b^2}\)
- **两点间向量**：\(\overrightarrow{AB}=\begin{pmatrix}x_B-x_A\\y_B-y_A\end{pmatrix}\)
- **中点**：\(\overrightarrow{OM}=\frac{\overrightarrow{OA}+\overrightarrow{OB}}{2}\)
- **三点共线**：\(\overrightarrow{AB}=\lambda\overrightarrow{AC}\)（或平行）
- **平行**：\(\mathbf{a}=k\mathbf{b}\)
- **垂直**：\(a_1b_1+a_2b_2=0\)（坐标点积为零）

**示例**：\(A(1,2),B(3,5),C(5,8)\)，\(\overrightarrow{AB}=(2,3),\ \overrightarrow{AC}=(4,6)=2(2,3)\)，共线。

**坑**：共线时 \(\lambda\) 可为负；垂直时坐标乘积和为零。

---

### 6.8.4 速度与合成（运动学应用）
- **速度向量**：\(\mathbf{r}(t)=\mathbf{r}_0+t\mathbf{v}\)（匀速直线运动）
- **相对速度**：\(\mathbf{v}_{A/B}=\mathbf{v}_A-\mathbf{v}_B\)
- **相遇条件**：位置向量相等（两个坐标同时相等）

**示例**：船 A 从 \((0,0)\) 以 \(\begin{pmatrix}2\\3\end{pmatrix}\) 行驶，船 B 从 \((10,0)\) 以 \(\begin{pmatrix}1\\4\end{pmatrix}\) 行驶，相遇？  
\(\mathbf{r}_A(t)=(2t,3t),\ \mathbf{r}_B(t)=(10+t,4t)\)。令 \(2t=10+t \Rightarrow t=10\)，此时 \(3t=30,\ 4t=40\)，不相等，故不会相遇。

**坑**：二维运动需两个坐标同时满足。

---

### 6.8.5 向量与直线、圆的结合
- 直线的方向向量：\(y=mx+c\) 的方向向量可设为 \(\begin{pmatrix}1\\m\end{pmatrix}\)。
- 直线的法向量：\(Ax+By+C=0\) 的法向量为 \(\begin{pmatrix}A\\B\end{pmatrix}\)。
- 圆的切线：圆心到切点的半径与切线垂直，用向量点积为零表示。

# Topic 7: 圆的坐标几何 (Coordinate Geometry of the Circle)

## 7.1 圆的方程

### 标准式
圆心 \((a,b)\)，半径 \(r\)：  
\[
(x - a)^2 + (y - b)^2 = r^2
\]

### 一般式
\[
x^2 + y^2 + 2gx + 2fy + c = 0
\]  
圆心 \((-g, -f)\)，半径 \(\sqrt{g^2 + f^2 - c}\)（要求 \(g^2+f^2-c>0\)）。

**配方法**：将一般式化为标准式，通过配方找出圆心和半径。

**示例**：\(x^2 + y^2 - 4x + 6y - 3 = 0\)  
\[
(x^2 - 4x) + (y^2 + 6y) = 3
\]  
\[
(x-2)^2 - 4 + (y+3)^2 - 9 = 3
\]  
\[
(x-2)^2 + (y+3)^2 = 16
\]  
圆心 \((2,-3)\)，半径 \(4\)。

**坑**：一般式需先整理，配方时注意加减平衡。

---

## 7.2 直线与圆的交点

### 方法：联立方程组
将直线方程（如 \(y = mx + c\)）代入圆方程，消去 \(y\)，得到关于 \(x\) 的一元二次方程：
\[
Ax^2 + Bx + C = 0
\]

**判别式 \(\Delta = B^2 - 4AC\) 决定位置关系**：

| \(\Delta\) | 几何意义 |
|-----------|----------|
| \(>0\) | 两个不同交点 → 直线是割线 |
| \(=0\) | 一个交点（重根）→ 直线是切线 |
| \(<0\) | 无交点 → 直线与圆相离 |

**若直线为垂直**（\(x = k\)），则代入得关于 \(y\) 的二次方程，同理。

### 求交点坐标
解二次方程得 \(x\)（或 \(y\)），代回直线方程得另一坐标。

### 切线方程（已知切点）
若切点为 \((x_0, y_0)\)，半径垂直于切线，斜率 \(m = -\frac{x_0 - a}{y_0 - b}\)（圆心 \((a,b)\)），切线方程为：
\[
y - y_0 = m(x - x_0)
\]

### 切线方程（从圆外一点引切线）
设切线斜率 \(k\)，方程 \(y - y_1 = k(x - x_1)\)，代入圆方程令 \(\Delta = 0\) 解 \(k\)，注意斜率不存在的情况。

**坑**：计算判别式时，要确保二次项系数不为零（直线不与圆相切于竖直方向时，直线代入后二次项系数一般非零）。

## 补充：圆的其他性质（教材可能分散或省略，但考试常考）

### 垂径定理
- 圆心到弦的垂线平分该弦。
- 若 \(O\) 为圆心，\(M\) 为弦 \(AB\) 中点，则 \(OM \perp AB\)。
- 应用：已知半径 \(r\) 和弦心距 \(d\)，弦长 \(= 2\sqrt{r^2 - d^2}\)。

**向量证明**：设 \(\overrightarrow{OM} = \frac{\overrightarrow{OA} + \overrightarrow{OB}}{2}\)，\(\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}\)，则 \(\overrightarrow{OM} \cdot \overrightarrow{AB} = 0\)。

### 圆与圆的位置关系
设两圆圆心距 \(d\)，半径 \(r_1, r_2\)：
- \(d > r_1 + r_2\)：相离
- \(d = r_1 + r_2\)：外切
- \(|r_1 - r_2| < d < r_1 + r_2\)：相交
- \(d = |r_1 - r_2|\)：内切
- \(d < |r_1 - r_2|\)：内含

### 公共弦方程
两圆方程相减，即得公共弦所在直线方程。
