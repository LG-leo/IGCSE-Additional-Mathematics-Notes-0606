# Topic 6: 直线图 (Straight-line Graphs)

## 6.1 线段的长度与中点

- **长度公式**：已知两点 A(x1, y1)、B(x2, y2)，线段 AB 的长度为 sqrt[(x2-x1)^2 + (y2-y1)^2]。
- **中点公式**：中点 M 的坐标为 ((x1+x2)/2, (y1+y2)/2)。
- **坑**：使用长度公式时，顺序无关，平方后开方取正。

---

## 6.2 平行与垂直

- **平行**：若两直线斜率存在，则 m1 = m2；若一条水平 (m=0)，另一条也水平。
- **垂直**：若两直线斜率存在，则 m1·m2 = -1；特殊：一条水平 (m=0)，一条垂直（斜率不存在）也垂直。
- **几何直觉**：平行 → 方向相同；垂直 → 斜率乘积 -1 或一水平一竖直。

###求2个线段的切率
A-B-C中，AB和BC有不同的斜率，那么：

(a_y-b_y)/（a_x-b_x）=(b_y-c_y)/（b_x-c_x）

---

## 6.3 直线方程

- **常见形式**  
  - 斜截式：y = mx + c（m 斜率，c 纵截距）  
  - 点斜式：y - y1 = m(x - x1)  
  - 两点式：由两点先求斜率 m = (y2 - y1)/(x2 - x1)，再代入点斜式。  
  - 一般式：Ax + By + C = 0（可求斜率 -A/B，截距 -C/B 等）
- **步骤**：  
  1. 确定已知条件（一点 + 斜率；两点；斜截距等）。  
  2. 选用适当形式写出方程。  
  3. 化为所需形式（如 y = mx + c 或 Ax + By + C = 0）。
- **坑**：垂直直线斜率不存在，方程应写为 x = a。

---

## 6.4 直线形面积（坐标系中的多边形面积）

- **三角形面积（已知三点）**：  
  设 A(x1,y1), B(x2,y2), C(x3,y3)，则  
  S = 1/2 * |x1(y2 - y3) + x2(y3 - y1) + x3(y1 - y2)|。  
  也可用行列式形式。
- **步骤**：将三点坐标代入公式 → 计算绝对值 → 除以 2。
- **示例**：A(1,2), B(4,5), C(6,1)  
  S = 1/2 * |1*(5-1) + 4*(1-2) + 6*(2-5)| = 1/2 * |4 -4 -18| = 9。
- **坑**：三点共线时面积为 0，可用于判断共线。
- **多边形面积**（鞋带公式）：  
  顶点按顺序排列，S = 1/2 * | Σ (xi·y(i+1) - x(i+1)·yi) |，其中 (x(n+1), y(n+1)) = (x1, y1)。

---

## 6.5 从非线性方程转换到线性形式

- **常用模型**：  
  - y = k·x^n：取常用对数 lg，设 Y = lg y, X = lg x → Y = n·X + lg k。  
  - y = k·b^x：取常用对数 lg，设 Y = lg y, X = x → Y = (lg b)·X + lg k。  
  - y = e^(px+q)：取自然对数 ln，设 Y = ln y → Y = p·x + q。  
  - y = a/x + b：设 X = 1/x, Y = y → Y = a·X + b。  
  - y = a·√x + b：设 X = √x, Y = y → Y = a·X + b。
- **步骤**：  
  1. 根据形式确定变换。  
  2. 计算新变量值。  
  3. 画图（或计算）得到直线斜率与截距。  
  4. 反推原方程参数。
- **坑**：取对数要求数据 >0；变换后直线斜率和截距对应原方程中的参数。

---

## 6.6 从线性形式反推非线性方程

已知变量 X, Y 满足 Y = m·X + c，且 X, Y 与原变量关系已知，求原方程。

- 若 Y = lg y, X = lg x → y = 10^c · x^m（幂函数）
- 若 Y = lg y, X = x → y = 10^c · (10^m)^x（指数函数）

这幂函数和指数函数模型的根本区别在于 **自变量 \(x\) 的位置**：

- 幂函数中自变量 \(x\) 在底数，指数函数中自变量 \(x\) 在指数。
- **模型1 \(y = k \cdot x^n\)**：\(x\) 在**底数**位置 → 取对数后，\(\lg x\) 与 \(\lg y\) 成线性关系，所以需要设 \(X = \lg x\)。
- **模型2 \(y = k \cdot b^x\)**：\(x\) 在**指数**位置 → 取对数后，\(x\) 与 \(\lg y\) 成线性关系，所以直接设 \(X = x\)。

**过程不一样，是因为它们要“拉直”的对象不同**：  
- 幂函数：\(\lg y\) 对 \(\lg x\) 作图是直线  
- 指数函数：\(\lg y\) 对 \(x\) 作图是直线

如果混用，就找不到正确的直线关系了。


- 若 Y = ln y, X = x → y = e^c · e^(m·x)（指数函数，底 e）
- 若 Y = y, X = 1/x → y = m/x + c（反比例）
- 若 Y = y, X = √x → y = m·√x + c（平方根型）

**注意**：系数 m, c 由直线读出，反推时注意底数（10 或 e）和定义域。

## 补充：非线性方程转线性形式的要点

### 1. 取对数的底数要看清
- 模型 \(y = kx^n\) 或 \(y = kb^x\)：用 **常用对数 \(\lg\)**（底数 10）  
  \[
  \lg y = n \lg x + \lg k
  \]
- 模型 \(y = e^{px+q}\)：用 **自然对数 \(\ln\)**（底数 e）  
  \[
  \ln y = p x + q
  \]
> **注意**：考题有时给的是 \(\log\) 没写底数，默认按常用对数处理；但如果方程里是 \(e^{...}\)，一定要用 \(\ln\)。

---

### 2. 变换后的“截距”要小心反推

| 原模型 | 变换式 | 斜率 | 截距 | 反推参数 |
|--------|--------|------|------|----------|
| \(y = kx^n\) | \(Y = nX + \lg k\) | \(n\) | \(\lg k\) | \(k = 10^{\text{截距}}\) |
| \(y = kb^x\) | \(Y = (\lg b)X + \lg k\) | \(\lg b\) | \(\lg k\) | \(b = 10^{\text{斜率}}\)，\(k = 10^{\text{截距}}\) |
| \(y = e^{px+q}\) | \(Y = px + q\) | \(p\) | \(q\) | 直接得到 \(p, q\) |

> **坑**：有些同学用 \(\ln\) 处理 \(y = kx^n\)，虽然理论上也可以（\(\ln y = n\ln x + \ln k\)），但考题如果给的数据表里用了 \(\lg\)，你就要保持一致，否则截距反推时底数会出错。

---

### 3. 作图法的步骤
考试中这种题经常分步设问：
1. 写出变换后的 \(Y\) 和 \(X\) 表达式  
2. 计算表格中缺失的 \(Y\) 或 \(X\) 值  
3. 画散点图，画最佳拟合直线  
4. 从图上读斜率、截距  
5. 反求原方程的参数  

> **注意**：读数时要保留足够精度，通常要求读到 0.01 或 0.1 级别，否则最后参数偏差大。

---

### 4. 数据必须为正
取对数要求数据 \(>0\)，如果表格里有 0 或负数，说明这个模型不适用——这一点题目有时会设计成“陷阱”。
---

## 6.7 从数据中寻找关系

- **方法**：  
  1. 绘制原始数据的散点图，观察趋势（直线、曲线、指数、幂函数）。  
  2. 猜测模型（如 y = k·x^n、y = k·b^x、y = a/x + b 等）。  
  3. 对猜测模型进行变换，得到新变量 X, Y。  
  4. 计算新变量，再绘制散点图，检查是否近似直线。  
  5. 若成直线，用直线上两点（或最小二乘法概念）求斜率 m 和截距 c。  
  6. 反推出原方程中的常数。
- **示例**：数据点怀疑 y = k·x^n，计算 X = lg x, Y = lg y，若散点近似直线，则 n 为斜率，lg k 为截距。
- **坑**：数据误差会导致点不完全在直线上，应画“最佳拟合直线”；取对数时注意零值或负值需排除。

**给出一组数据 \((x, y)\)，用数据变换找出 \(y\) 与 \(x\) 的关系式**：这里有一个通用的判断和操作流程：

---

### 1. 根据数据形态猜测模型
- 若 \(y\) 随 \(x\) 增长越来越快（指数型） → 试 \(y = k b^x\) 或 \(y = e^{px+q}\)
- 若 \(y\) 与 \(x\) 成幂次关系（如抛物线、反比等） → 试 \(y = k x^n\)
- 若 \(y\) 与 \(1/x\) 成线性 → 试 \(y = a/x + b\)
- 若 \(y\) 与 \(\sqrt{x}\) 成线性 → 试 \(y = a\sqrt{x} + b\)

---

### 2. 计算新变量，看哪个成直线
对每个候选模型，构造对应的 \(X, Y\)，然后：
- 画散点图看是否近似直线，或
- 计算相关系数（若题目允许），或
- 根据题目提示（常会直接说“已知 \(y\) 与 \(x\) 满足 \(y = k b^x\)”）

---

### 3. 求直线斜率和截距
用线性回归公式或直接从图上读两点：

\[
\text{斜率} = \frac{Y_2 - Y_1}{X_2 - X_1}, \quad \text{截距} = Y - \text{斜率} \cdot X
\]

---

### 4. 反推原方程参数
根据变换式回代：

| 模型 | 变换 | 斜率 | 截距 | 反推 |
|------|------|------|------|------|
| \(y = k x^n\) | \(Y=\lg y,\; X=\lg x\) | \(n\) | \(\lg k\) | \(k=10^{\text{截距}}\) |
| \(y = k b^x\) | \(Y=\lg y,\; X=x\) | \(\lg b\) | \(\lg k\) | \(b=10^{\text{斜率}},\; k=10^{\text{截距}}\) |
| \(y = e^{px+q}\) | \(Y=\ln y,\; X=x\) | \(p\) | \(q\) | 直接得到 \(p,q\) |
| \(y = \frac{a}{x}+b\) | \(X=1/x,\; Y=y\) | \(a\) | \(b\) | 直接得到 \(a,b\) |
| \(y = a\sqrt{x}+b\) | \(X=\sqrt{x},\; Y=y\) | \(a\) | \(b\) | 直接得到 \(a,b\) |

---

### 例子（简略）
数据：

| \(x\) | 1 | 2 | 3 | 4 |
|------|---|---|---|---|
| \(y\) | 2.5 | 6.3 | 15.8 | 39.7 |

发现 \(\lg y\) 与 \(x\) 几乎成直线 → 用 \(y = k b^x\)。  
计算得斜率 \(\approx 0.477\) → \(b = 10^{0.477} \approx 3.0\)；截距 \(\approx 0.398\) → \(k = 10^{0.398} \approx 2.5\)。  
所以关系式为 \(y = 2.5 \times 3^x\)。

# 6.8 向量基础与几何应用（纯文本版）

---

## 一、向量的表示

### 1. 两种常见形式
- 列向量：写成竖排，例如 (3,4) 的列形式
- 单位向量组合：a i + b j，其中 i=(1,0)，j=(0,1)

考试中两种形式均可，按题目要求写。

### 2. 有向线段
向量 AB 表示从 A 指向 B 的向量，其坐标为 (xB - xA, yB - yA) = 向量 OB - 向量 OA。
注意：向量 BA = 向量 OA - 向量 OB，方向相反。

---

## 二、位置向量与单位向量

### 1. 位置向量
点 P(x,y) 的位置向量是从原点 O 到 P 的向量，记作 OP = (x, y)。
任意向量 AB = OB - OA。

### 2. 单位向量
与向量 a 同方向且长度为 1 的向量：a_hat = a / |a|，其中 |a| = sqrt(a1^2 + a2^2)。
反向单位向量： -a / |a|。
示例：a = (3,4)，|a| = 5，单位向量 = (3/5, 4/5)。

---

## 三、向量的基本运算

### 1. 加法
代数：(a,b) + (c,d) = (a+c, b+d)
几何：三角形法则（首尾相接）或平行四边形法则。

### 2. 减法
a - b = a + (-b)
几何：从 b 的终点指向 a 的终点（起点重合时）。
坐标：AB = OB - OA 正是减法。

### 3. 数乘
k·(a,b) = (k·a, k·b)
k>0：方向相同，长度变为 |k| 倍；k<0：方向相反；k=0：零向量。

### 4. 模长
|a| = sqrt(a^2 + b^2)

### 5. 中点公式
若 M 是 AB 中点，则 OM = (OA + OB)/2
坐标：M((xA+xB)/2, (yA+yB)/2)

---

## 四、平行、共线与垂直

### 1. 平行
a 与 b 平行当且仅当 a = k·b（k ≠ 0）。
坐标形式：a1/b1 = a2/b2（分母不为 0 时）。

### 2. 三点共线
A,B,C 共线 <=> 向量 AB = λ·向量 AC（或 AB 与 BC 平行）。λ 可以为任意非零实数（负数也可）。
示例：A(1,2), B(3,5), C(5,8)，AB = (2,3), AC = (4,6) = 2·(2,3)，所以共线。

### 3. 垂直（点积）
垂直条件用坐标点积：a·b = a1·b1 + a2·b2 = 0。
几何意义：|a||b|cosθ = 0 → θ = 90°。
示例：a=(2,3), b=(3,-2)，2·3+3·(-2)=0，垂直。

---

## 五、运动学应用（速度与合成）

### 1. 匀速直线运动的位置向量
r(t) = r0 + t·v，其中 r0 是初始位置，v 是速度向量。

### 2. 相对速度
物体 A 相对 B 的速度：v_A/B = v_A - v_B。

### 3. 相遇条件
存在同一个 t >= 0 使 r_A(t) = r_B(t)，即两个坐标分量分别相等。
经典例题：船 A 从 (0,0) 以速度 (2,3) 出发，船 B 从 (10,0) 以速度 (1,4) 出发，问是否相遇？
解：r_A(t) = (2t, 3t)，r_B(t) = (10+t, 4t)
由 2t = 10+t 得 t=10，此时 y_A=30, y_B=40，不相等，故不会相遇。

### 4. 距离最近问题（进阶）
若不相遇，求何时距离最近。
设相对位置向量 r_AB(t) = r_A(t) - r_B(t)，距离平方是 t 的二次函数，求最小值点即可。

---

## 六、向量与直线、圆的结合

### 1. 直线的方向向量
- 斜截式 y = mx + c：方向向量可取 (1, m)
- 一般式 Ax + By + C = 0：方向向量可取 (B, -A)（因为与法向量垂直）

### 2. 直线的法向量
直线 Ax + By + C = 0 的法向量 n = (A, B)，与直线垂直。

### 3. 圆与切线
- 圆心 C，切点 T，半径 CT 与切线垂直。
- 若切线方向向量为 d，则 CT·d = 0。
示例：圆 x^2+y^2=25，点 P(3,4) 在圆上，圆心 O(0,0)，半径向量 (3,4)，切线方向与它垂直，可取 (4,-3)，切线方程过 P 可求得。

---

## 七、常见综合题型及解法

| 题型 | 方法要点 |
|------|----------|
| 几何证明（平行四边形、共线等） | 利用向量相等、数乘关系、中点公式 |
| 坐标运算（求向量、模、单位向量、参数） | 直接套用定义，联立方程 |
| 运动学（相遇、最近距离） | 列位置向量方程，解时间 t，或用二次函数最值 |
| 直线与圆（切线、垂直） | 用垂直条件（点积为 0）列方程 |

---

## 八、易错点速查表

| 易错点 | 正确做法 |
|--------|----------|
| 向量 AB 写反 | 终点减起点：OB - OA |
| 三点共线忽略 λ 可为负 | 平行即可，方向可相反 |
| 垂直条件误用成数乘 | 垂直用点积 = 0；平行用数乘 |
| 运动相遇只列一个坐标方程 | 两个坐标必须同时满足，t 相同 |
| 单位向量忘记除以模长 | a_hat = a / |a| |

# Topic 7: 圆的坐标几何 (Coordinate Geometry of the Circle)

## 7.1 圆的方程

- **标准式**：圆心 (a,b)，半径 r：(x - a)² + (y - b)² = r²
- **一般式**：x² + y² + 2gx + 2fy + c = 0  
  圆心 (-g, -f)，半径 sqrt(g² + f² - c)（要求 g²+f²-c>0）。
- **配方法**：将一般式化为标准式，通过配方找出圆心和半径。  
  **示例**：x² + y² - 4x + 6y - 3 = 0  
  (x²-4x) + (y²+6y) = 3 → (x-2)² -4 + (y+3)² -9 = 3 → (x-2)² + (y+3)² = 16  
  圆心 (2,-3)，半径 4。
- **坑**：一般式需先整理，配方时注意加减平衡。

---

## 7.2 直线与圆的交点

- **方法**：联立方程组  
  将直线方程（如 y = mx + c）代入圆方程，消去 y，得到关于 x 的一元二次方程：Ax² + Bx + C = 0。
- **判别式 Δ = B² - 4AC 决定位置关系**：  
  - Δ > 0：两个不同交点 → 直线是割线  
  - Δ = 0：一个交点（重根）→ 直线是切线  
  - Δ < 0：无交点 → 直线与圆相离  
  若直线为垂直（x = k），则代入得关于 y 的二次方程，同理。
- **求交点坐标**：解二次方程得 x（或 y），代回直线方程得另一坐标。
- **切线方程（已知切点）**：  
  若切点为 (x0, y0)，半径垂直于切线，斜率 m = -(x0 - a)/(y0 - b)（圆心 (a,b)），切线方程为 y - y0 = m(x - x0)。
- **切线方程（从圆外一点引切线）**：  
  设切线斜率 k，方程 y - y1 = k(x - x1)，代入圆方程令 Δ = 0 解 k，注意斜率不存在的情况。
- **坑**：计算判别式时，要确保二次项系数不为零（直线不与圆相切于竖直方向时，直线代入后二次项系数一般非零）。

---

## 补充：圆的其他性质（教材可能分散或省略，但考试常考）

### 垂径定理
- 圆心到弦的垂线平分该弦。  
  若 O 为圆心，M 为弦 AB 中点，则 OM ⊥ AB。
- 应用：已知半径 r 和弦心距 d，弦长 = 2·sqrt(r² - d²)。
- **向量证明**：设 OM = (OA+OB)/2，AB = OB - OA，则 OM·AB = 0。

### 圆与圆的位置关系
设两圆圆心距 d，半径 r1, r2：
- d > r1 + r2：相离
- d = r1 + r2：外切
- |r1 - r2| < d < r1 + r2：相交
- d = |r1 - r2|：内切
- d < |r1 - r2|：内含

### 公共弦方程
两圆方程相减，即得公共弦所在直线方程。

---

## 总结：关键公式表

- 斜率：m = (y2 - y1)/(x2 - x1)
- 中点：((x1+x2)/2, (y1+y2)/2)
- 距离：sqrt((x2-x1)² + (y2-y1)²)
- 三角形面积（坐标）：1/2 * |x1(y2-y3) + x2(y3-y1) + x3(y1-y2)|
- 圆的标准方程：(x-a)² + (y-b)² = r²
- 圆的一般方程圆心：(-g, -f)
- 圆的一般方程半径：sqrt(g² + f² - c)
- 直线与圆位置：联立 → 判别式
- 垂径定理：弦长 = 2·sqrt(r² - d²)
